# MCP Guidance Guide

## Tier 1 — Start here:

- **sequential-thinking** — to break down the page structure, decide on layout system, component hierarchy
- **exa/get_code_context_exa** — to find current best practices for minimal, essay-style landing pages in Next.js

## Tier 2 — Design/docs lookup:

- **shadcn** — check for any useful primitives (maybe just button for CTA, tooltip/popover for the clickable easter eggs)
- **tailwind** — for typography scale, spacing, max-width, responsive patterns
- **next-devtools/nextjs_docs** — if any Next.js-specific routing or config questions come up

## Tier 3 — Verification:

- **nextjs_index/nextjs_call** — to verify no runtime errors once running

## Tier 4 — Deploy:

- **vercel** — when ready to ship
